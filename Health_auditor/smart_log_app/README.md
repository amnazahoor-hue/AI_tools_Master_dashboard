# Smart-Log Health & Productivity Auditor

A 100% free local project using Flask + SQLite + NumPy + Tailwind + Chart.js.  
No paid APIs, no cloud AI dependencies.

## Structure

- `backend/` Flask API, SQLite DB, and local AI logic
- `frontend/` Dashboard and Admin UI

## Setup

1. Open terminal in `smart_log_app/backend`
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Start server:
   - `python app.py`

## Access

- User dashboard: `http://127.0.0.1:5000/`
- Admin panel: `http://127.0.0.1:5000/admin`
- Health check: `http://127.0.0.1:5000/health`
